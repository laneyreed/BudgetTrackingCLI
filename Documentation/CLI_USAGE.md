# Budget Tracker CLI - User Guide

### Prerequisites:
- Python 3.7 or higher
- `rich` library (for beautiful CLI formatting)

### Install Dependencies:
- `pip install rich`

## Step-by-Step

- **Step 1:** Launch the Application
- `python run_cli.py`
- **Step 2:** Create Your First User
- **Step 3:** Create Your First Account
- **Step 4:** Add Your First Transaction
- **Step 5:** View Your Transactions
- **Step 6:** View Financial Summary

### Navigation Tips:
- Type the number of your choice and press Enter
- Press `Ctrl+C` to exit at any time
- Use `0` to go back to previous menu
- All prompts show available options

## Keyboard Shortcuts

- **Ctrl+C**: Exit the application at any time
- **Enter**: Confirm selection or continue
- **0**: Return to previous menu (in most screens)
